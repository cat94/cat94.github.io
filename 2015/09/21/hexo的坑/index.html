    • hexo的坑

    hexo很酷，但是在使用过程中也遇到了很多坑…在此记录一下这些或大或小的坑。

    1. hexo中使用LaTeX语法，由于下划线”_“会先被解析为强调符号，所以需要将LaTeX的下标”_“改成”\_“，然后本地就没办法看到公式效果了= =

    2. 标题#后一定要加空格，否则无法解析…

    3. 公式中不能有中文…..

    4. 博客部署到github/gitcafe上,config中deploy的type:github需要改成git,然后安装插件，否则会部署失败：
      hexo ERROR Deployer not found: github

    5. 在自己的Mac系统上安装插件失败:npm install hexo-deployer-git —save，应该是权限问题，使用如下方式解决:
      sudo npm install —unsafe-perm —verbose -g hexo
